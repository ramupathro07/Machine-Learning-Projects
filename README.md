# 🚀 Machine Learning Projects Portfolio

## 📌 Project 1: Spam SMS Classification

### Overview
A Machine Learning project that classifies SMS messages as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques and classification algorithms.

### Goal
- Detect spam messages automatically.
- Improve communication security.
- Compare multiple ML algorithms for best performance.

### Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

### Output
- SMS messages classified as Spam or Ham.
- Performance comparison of multiple models.
- Confusion Matrix and Classification Report.

### Accuracy
✅ Random Forest: **99% Accuracy**

## 📌 Project 2: Sentiment Analysis of Restaurant Reviews

### Overview
A Natural Language Processing (NLP) project that analyzes restaurant reviews and predicts customer sentiment as **Positive** or **Negative**.

### Goal
- Understand customer feedback.
- Automate sentiment classification.
- Support business decision-making using review insights.

### Tools & Technologies
- Python
- Pandas
- NumPy
- NLTK
- Scikit-Learn
- CountVectorizer (Bag of Words)
- Jupyter Notebook

### Output
- Reviews classified as Positive or Negative.
- Sentiment prediction based on customer comments.
- Evaluation using classification metrics.

### Accuracy
✅ Naive Bayes Model: **78.5% Accuracy**
